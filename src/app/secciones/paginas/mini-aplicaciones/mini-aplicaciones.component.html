Aplicaciones desarrolladas en cursos online

En esta aplicación aprendí a trabajar de forma modularizada, injectar servicios en mis proyectos y a consumir información de una API (en este caso una de GIFs). El listado del lado izquierdo de la web, es de un máximo de 10 items y, cada búsqueda realizada, se mantiene en ese menú porque utilicé localStorage.

IMPORTANTE: Estás primeras aplicaciones NO SON RESPONSIVE.

Host Proyectos: Netlify

API SERVER: GIPHY Developers

Tecnologías Utilizadas: Angular, Typescript, Animate CSS, Bootstrap.
